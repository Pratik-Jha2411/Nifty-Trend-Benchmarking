Financial Sentiment Analysis using FinBERT, BERT, VADER & Naive Bayes

A comprehensive NLP-based financial sentiment analysis system that compares the performance of multiple sentiment analysis models including FinBERT, BERT, VADER, and Naive Bayes for classifying financial news and market-related text into positive, neutral, and negative sentiments.

This project implements- Financial text preprocessing and cleaning, Transformer-based deep learning models, Lexicon-based sentiment analysis, Traditional machine learning classification, Model training and evaluation pipelines, Comparative performance analysis and visualization

Features- Financial news sentiment classification, FinBERT and BERT transformer implementation, VADER lexicon-based sentiment scoring, Naive Bayes ML classifier using TF-IDF, Accuracy, Precision, Recall, and F1-score evaluation, Confusion matrix and visualization support, Comparative benchmarking across models, End-to-end NLP workflow implementation

Tech Stack- Python, PyTorch, Transformers, Scikit-learn, NLTK, Pandas & NumPy, Matplotlib & Seaborn

Applications- Stock market trend analysis, Financial news monitoring, Investor sentiment analysis, Market prediction research, NLP-based financial analytics

Repository Structure- FinBERT_sentiment_analysis.ipynb, BERT_Sentiment_Analysis.ipynb, VADER_sentiment_analysis.ipynb, NAIVE_BAYES_sentiment_analysis.ipynb, Output

The system predicts- Positive Sentiment, Neutral Sentiment, Negative Sentiment, while identifying the best-performing sentiment analysis model for financial datasets
